# My Microservice Project

This repository contains a sample CRUD application with a .NET backend and a React frontend.

## Project Structure
- `Backend/`: Contains the .NET API project for CRUD operations.
- `Frontend/`: Contains the React application.
- `docker-compose.yml`: Configuration for local development with Docker.

### Getting Started

1. Clone the repository.
2. Run `docker-compose up` to start the services.
